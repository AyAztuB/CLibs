Detailed Description

Macros for defining 64-bit literals portably.

Maximum and minimum values for integer types.

Note
Signed Max(iX) = (iX)(((uX)(~0)) >> 1) and Min(iX) = -Max(iX)-1 = 1 << (X-1)
Unsigned Max(uX) = (uX)(-1) = (uX)(~0) = (((uX)(Max(iX))) << 1) + 1
